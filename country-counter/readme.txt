== Description ==
 
Display the percentage and graph of countries of the world you have been to in a widget.
 
[Feedback](https://wordpress.org/plugins/country-counter)
 
== Installation ==
1. Unpack the `download-package`.
2. Upload the file to the `/wp-content/plugins/` directory.
3. Activate the plugin through the `Plugins` menu in WordPress.
4. Go to Appearance -> Widgets Section -> Select Plugin `Country Counter Widget`.
5. Done and Ready.
